# Express NextJs - Custom Server

Tailwind css is also used as a CSS framework.

## Getting Started

> First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result

> To build a project:
```bash
npm run build
# or
yarn build
```

> For production :

```bash
npm run start
# or
yarn start
```
